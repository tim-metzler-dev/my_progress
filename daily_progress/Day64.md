## Today I learned:
  1. You can set a image as background. Your elements will be on the image like backgroundcolor. The CSS property is:
       background-image: here you type in the whole url;
  2. I have a small corecction. A couple of days ago I said, that max-width ... is only imporrtant for responsive webdesign. But you can and should also use this, when you have elements with space between them.
     If the space is depending on the screen size (if you set width... to %) it could be better for people with bigger screens seeing your content.
  3.  You can change the font family if you type font-family: ...;   I will learn later how to add font familys from google fonts.
          For examples I could name: sans serif, Impact, italic
  4. You can add a backup font with adding two font types to the font family seperated by ",". If the first font isn't avaible the second one will be used. But if it is avaible (the first) it always will use the first.
  5. I learned about borders. Borders are the elements at the end of an element. The lines. You can change the color with border-color.
  6. There also is font-size. It sets the size of the fonts and works like every typical css element.
