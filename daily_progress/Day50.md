## Today I learned:
	1. The meta viewport tag: It's crucial to use this element for responsive web design. You can use it also with only HTML code. 
     I will add this to my boilerplate to not think about when and where I would have to use it:
		<meta name="viewport" content="width=device-width, initial-scale=1.0">
		
			The width=device-width tells the browser to set the width of the page to match the screen width of the device.
			The initial-scale=1.0 sets the initial zoom level when page is first loaded. That means, if you load your page it's instant by zoom 100%. 1.0 means 100%.
      Of course you can also change the value.
			Ther is also the user-scalable=no tag is recomended to avoit, becauseof accessebility reasons.
