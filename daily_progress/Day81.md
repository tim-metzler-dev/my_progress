## Today I learned:
  1. We are going to learn something about background images>
       I. To set a backgroundimage you can use "    background-image: url("   with  your    src    ");"
      II. To change the size of the background images you can use background-size:
           a. contain    -  scales the image as large as possible without cropping or stretching
           b. cover      -  covers the whole background so that you can't see anything. The picture often will be to big and things will be cut away.
     III. To set if the image should be repeated, that means having much of the same image covering everything, you can use background-repeat  . Again this has some options:
           a. repeat-x/y - You only say x or y. But if you set it to x it will repeat only horizontally and if you set y it will only repeat vertically.
           b. no-repeat  - Yeah. I think that's understandable. The picture won't be repeated.
      IV. To set where the image should be in the screen you can use background-position. You have clear options:   top, bottom, left, right, and center, or specific pixel or percentage values.
             If you set like top left you just write it as I did without ",".
       V. If you want to set if a background image is static or will move with the content you can use background-attachment. This has two options:
           a. scroll     - This is the default one. It will set the picture to be moved with the content.
           b. fixed      - This will make the background image won't move when you move content.

      VI. If you want to type in everything at once it will work like this: {background: background-image, background-position, and background-attachment; }
