## Today I learned:
  1. The line-height property. It is used for text-elements. It always applies when a text wraps (if you have a long p it also wraps.
       eventhough it hasn't br it also will be able to use line-height). If you set a number the space will be the number x font-size.
  2. The list-style-type property. It chenges the first thing in a list. The circles or numbers (ul/ol)
       I. By ul:
         you have discs, circles, or squares.
      II. By ol:
         you have different numbering systems, like decimal, Roman numerals, or even alphabetical characters.
               you can try all these out. Some names also tell what it does.
  3.  The list-style-position property. You only have two values: ind´side and outside and I'll try to eplain them as good as I can.
         The outside value is the default one.
          The inside not. If you set it to inside, when the text wraps in your li the number (ol) / circle (ul) will be there where the wraped tex starts.
          So the wrapped part is more left then the other part.   -- Another description-- :
            And, when you use the value inside, the bullet or number appears inside the content, which may cause the text to wrap and align with the bullet or number.
  4. The last one is an element that isn't used often because it isn't flexible list-style-image. It makes a picture over the li. It is used like this:
               list-style-image: url('...');
