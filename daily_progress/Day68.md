## Today I learned:
  1. If you have temporary changes in a style you can use the !important keyword. It overrides every other element and even overrides inline.
       {property: value !important;}
  2. The order of appereance in CSS: When two rules have the same specificity, the one that appears last in the CSS will be applied.
