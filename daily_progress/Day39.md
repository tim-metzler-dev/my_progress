## Today I learned:
	1. About ARIA roles. They are there to give an element more semantic informations. For example a button has a semantic meaning. But a div don't have one and screen readers don't know what's it function. But you can set a role="what you will define". But keep in mind that setting role attributes don't add any extra functions
	2. Role attributes are there to define elements that don't exist in semantic html.
