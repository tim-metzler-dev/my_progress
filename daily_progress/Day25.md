## Today I learned:
	1. The fieldset tag. It's used to make a square around the elements to show that they have own meaning and so on. For description (like the label element) you use legend:
		<fieldset>
		<legend>Name it as you like!</legend><input …></fieldset>
	2. You use size="20" to set a value in input how much letters are going to be shown/seen.
In type="number" of input you can set min and max. That will set a minimum age that you can write and a maximum age.
