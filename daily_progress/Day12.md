## Today I learned:
	1. The different target attributes.
		a. _self is the default setting. It opens the link in your tab.
		b. _blank . We know it. It opens the link in a new window.
		c. _parent. This opens the link in parent frame (if you put it in iframe) in the frame.
		d.  _top. This opens the link if you do it in an iframe in the whole page (like usually _self if not in iframe
		e. _unfencedTop. It's like a _top but for high security frames.
	2. The link of the anchor tag  <a> has different states.
		a. :link this is the state how the link usually is if you don't interacted with it.
		b. :visited this is the state that you see after you have visited that site.
		c. :hover describes what happens if you hover with your cursor over the link
		d. :focus happens when you focus the link
		e. :active is the state when you clicked the link but not stopped clicing.
