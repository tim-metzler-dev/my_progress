## Today I learned:
	1. In tables you can have th also in the body in a tr tag with td tags. 
	2. To make sure screen readers can definetifly read you table, you should use scope="x" and for x you either could use x="col" or "row" to define if your th is for a col or in a row. 
	3. You also shouldn't define width and height for table cells because people can't adjust the textsize that easy.
	4. You can use rowspan="…" and it's the same as colspan with the only difference that it goes horizontal and not vertical.
	5. Let's talk a bit why labels are good for inputs. At first screen readers have it easier and it is better for people with dissabilitys. But it also helps search engines to understand the content of your page better.
