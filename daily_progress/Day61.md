## Today I learned:
  1. You comment things in CSS with /* here is the content */   . As a reminder in HTML we do <!-- ... -->   .
  2. You also can set the margin/padding to auto. This sets the element in the perfect hotizontal center of the page. It doesn't work vertically even with ...top/bottom.
  3. You can set width/height ... to %. This will set the thing that you wrote with the width... to the % value of the size it has (the container). This is the easiest type of responsive webdesign.
