## Today I learned:
	1. The aria-describedby attribute. It's mostly used in associate instructions and error messages with form inputs. It describes an element with an input id.
	2. The alt attribute in images is important for people with visual disabilitys, so that they know what is in that image.
		Only important images that aren't decorative only should have an alt attribut with text.
		Every image should have an alt attribute even if it's empty. So that the screen reader don't reads the url of the image but the alt.
		Alt shouldn't be just one worded elements. They should include much informations, but not every detail only the main ones.
	3. About link texts. So at first you shouldn't write down only the action but also the destination. Like 
	Get details about our upcoming webinar     instead of:
Details.     That helps users with disabilitys to navigate better trough the site. But it's also helpfull for every other individual visiting your website.
