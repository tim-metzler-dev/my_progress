## Today I learned: 
  1. Layout: It is like a blueprint for design. It sais how elements are proportioned and everything is placed. It focuses on the placement of elements.
  2. Alignment: How elements relate to each other. For text but also images and more.
  3. Composition is the art of arranging elements to create a harmonious design. It determines how elements relate to each other and contribute to the design in an artistic way.
  4. Balance: How the visual weight is placed in a composition.
  5. Hierachie: Defines the order of importance of elements in a design.
  6. Contrast: How something is other from something else.
  7. White space: It is the free space like the japanese concept ma. So it don't even has to be white. It can be any color!
  8. UI(user interface): This is where human interacts with computer. It is every visual and interactive element.
  9. UX(user experience): This is how user feels while using the site. To help it your site could be accessible, easy to use and enjoyable.
