## Today I learned:
	1. To add an external CSS file to your HTML code you use the link element:
		<link rel="stylesheet" href="styles.css">
		- In this example the file is style.css but of course you will have to add other css files too.
	2. The basic CSS syntax is like this:
		selector {
  property: value;}
			The selector is the element you want to style. In 3. you will have more informations.
			There are much propertys. For example: Background-coulor, width, heigt…
	3. We have three types of selectors:
		a. The id. For an id you write down #nameofid it starts everytime with an #
		b. The class. For a class you will write down a dot: .this_class
		c. The tag. You also can write down an tag name like p. It's just p for selector.
	4. To add multiple elements as selector you use an comma (,) to seperate.
