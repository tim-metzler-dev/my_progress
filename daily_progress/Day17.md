## Today I learned:
	1. The sup tag. It's used to make a letter above the baseline like 2² is 4. In this case:
		<p>2<sup>2</sup> is 4</p>
	2. The sub tag. It's used to make a letter under the baseline like in CO2. In this case:
		<p>CO<sub>2</sub></p>
	3. The code element. It's used to show short codelines in html.<code></code>
	4. The pre tag. It's used to represent preformatted text. So when you write down something be mindfull that it will show every space and letter 1:1.
		You can use <pre><code>to display a code with more then one line</code></pre>
