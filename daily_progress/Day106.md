## Today I learned:
  1. Infinite scrolling is something used in social media plattforms like Instagram. This shows endless long more content as the user scrolls down.
  2. Pagination is like you are on a google site but have to turn to the next page to see more content.
  3. A good practice for infinity mode is adding a load more button. So can the user decide when or if he wants more content.
  4. Another good practice is adding a back to top button to improve user experience. This helps because the user don't has to scroll everything up again.
  5. If you use infinity mode without a load more button you should concidder adding a loading... text because without it some users may think the website is broken.
  6. The last but not least practice is to keep a footer accesible if it contains important informations. Because the  user should have access to links and important information all the time.

  7. A modal is a popup window that opens on top of the page and demands users attention before thay can continue.
  8. The content behind the modal is yousually dimmed so the modal gets the attention.
  9. You have commonly 2 buttons on a modal. A CAL(call to action) (like subrice and so on)and a close button.
