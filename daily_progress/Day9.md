## Today I learned:
	1. If you use src in the video tag you don't have a backup. But it's faster if you only have one format. But if you have more formats you can use them as Backups. For this you will need the source tag. Like this (and how you name the types of different formats.
		 <video controls>
			    <source
			      src="link to image with .mp4"
			      type="video/mp4"
			    >
			    <source
			      src="link to image with .webm"
			      type="video/webm"
			    >
			    <source
			      src="link to image with .ogg"
			      type="video/ogg"
			    >
			    <source 
			    src="link to image with .mov"
			    type="video/quicktime"
      </video>
