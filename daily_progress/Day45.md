## Today I learned:
	1. You should name your files so, that people can understand what is in that file.
	2. Every website has a root. You have assets. This is where you put all the images and fonts. You also have css and than html things. Many roots have a special README.md. This is a place, that provides additonal usefull informations. Index.html is the main site.
		It could look like this:
		.
├── /assets
│   ├── /images
│   │   ├── logo.png
│   │   └── icons.svg
│   ├── /fonts
│   │   ├── custom-font.woff
├── /css
│   ├── main.css
├── index.html
├── about.html
├── contact.html
└── README.md
		Of course a reel website has much more elements but this is an easy example.
